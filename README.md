Pothole Locator
=====================

A project using PhoneGap and AngularJS (with routing!)

## Install


Directory structure:

```
/PotholeLocator
    /css
        - index.css
    /img
        - logo.png
    /js
        - app.js
        - controllers.js
        - routers.js
    /lib
        - angular-1.0.5.js
        - cordova-2.5.0.js
    /partials
        - account.html
        - history.html
        - tag.html
    /res
        - whatever is in res
```

This is a work in progress
